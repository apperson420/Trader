# AI Coaching and Approval Boundaries

Trader Command Center can use software intelligence to coach, research, score, simulate, validate, and draft plans.

The safe end product is an AI-guided command center, not an unattended money-moving system.

## What the app may do

- explain market and chart evidence
- teach the user what each control means
- score setup quality
- compare plans against journal history
- run paper-only experiments
- draft a proposed action for user review
- show risk, assumptions, downside, and missing evidence
- log every step for review

## What must require human confirmation

Any action that can affect a real brokerage account must require the user to review and approve it through the existing manual ticket gates.

Required gates include owner access when configured, server-side environment configuration, a small risk cap, visible account check, human-review checkbox, exact typed confirmation, audit log, and emergency lock controls.

## What is not allowed

The app must not run unattended real-account actions from AI Coach, Safe Autopilot, Guided Workflow, Strategy Evolution, Validation Forge, AI Brain, or background workflows.

## Product direction

Build intelligence first in paper mode. Then allow AI-assisted review for real-account mode only with explicit user approval each time.

This app is not investment advice and does not guarantee profit.
